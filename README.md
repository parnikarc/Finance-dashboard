The **Finance Analytics Dashboard** is an interactive Power BI reporting solution designed to analyze sales, profit, product performance, and country-level business insights. The dashboard consolidates multi-year retail data into an intuitive interface, enabling stakeholders to make informed, data-driven decisions across segments, products, and regions.

This project includes three major report pages:
- **Finance Dashboard Overview**
- **Product Performance Analysis**
- **Country Performance Analysis**

Each page contains filters for **Year, Month, and Product**, supporting dynamic and flexible exploration of business metrics.
